> 2. 第一个以信鸽为载体的 IP 网络标准的 RFC 文档中推荐使用的 MTU (Maximum Transmission Unit) 是多少毫克？  
提示：咕咕咕，咕咕咕。

啊，著名[愚人节 RFC](https://tools.ietf.org/html/rfc1149)。里面提到：

> A typical MTU is 256 milligrams.

虽然并不是<b>推荐</b>，但估计答案是他了。
